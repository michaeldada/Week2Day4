Arrays

JavaScript arrays are used to store multiple values in a single variable. They can have just about anything as elements. Strings, numbers, booleans, more arrays, variables, and more(?). 

            var myLetters = ["a", "b", "c", "d"];
            var myNumbers = [1, 2, 3, 9];

Press the button to see an array of strings appear. This button will show you some AWESOME array methods.

Want to know how I did this? That big fat array is everything that we wanted to appear as a string. We used a .forEach() method to append each value in the array in our un ordered list with an id = "ex-methods"

            $(document).ready(function() {
              $("button#example").click(function() {

                var methods = ['<li>pop(): remove the last element</li>', '<li>push(): add an element to the end</li>', '<li>concat(): combine with another array</li>', '<li>reverse(): reverse the order of the array</li>'];

                methods.forEach(function(method){
                  $("#ex-methods").append(method);
                });
              });
            });
