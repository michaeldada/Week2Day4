JQuery Branching

Very often when you write code, you want to perform different actions for different decisions. You can use conditional statements in your code to do this.In JavaScript we have the following conditional statements:

  • Use if to specify a block of code to be executed, if a specified condition is true
  • Use else to specify a block of code to be executed, if the same condition is false
  • Use else if to specify a new condition to test, if the first condition is false

Click the button below and enter your age


You are an adult.

You are a baby.

How does this work? When the button is pressed, the user is prompted with a question from Master Computer. Master Computer determines whether or not you are a baby or an adult. If you are 21 or over, then you are an adult. Else, you are a baby. 

            Our JS looks like

            $(document).ready(function() {
              $("button#example").click(function() {
                var age = parseInt(prompt("Enter your age, human."));

                if (age >= 21) {
                  $(".show21").show();
                } else {
                  $(".show20").show();
                }
              });
            });
            --------------------------------------------
            Our CSS looks like

            .show21, .show20 {
              display: none;
              font-size: 300%;
            }
