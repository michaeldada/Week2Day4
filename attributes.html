JQuery Attributes

jQuery gives us the means to easily manipulate an element's attributes and gives us access to the element so that we can also change its properties. Lets walk through some examples below.

How does this work? The button is toggling a class called "blueFont" for all "p" tags. 

            $(document).ready(function() {
              $("button#example").click(function() {
                $("p").toggleClass("blueFont");
              });
            });
