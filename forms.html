Using JQuery with Forms

HTML forms are used to collect user input. HTML forms contain form elements. Form elements are different types of input elements, checkboxes, radio buttons, submit buttons, and more. The <\input> element is the most important form element. The <\input> element has many variations, depending on the type attribute. To see more about this visit HTML Forms W3 Schools

Once again, an HTML form is used to collect user input. Then we can use jQuery to target and manipulate the data
